# Brendan-nasa Brew

## How do I install these formulae?

`brew install brendan-nasa/brew/<formula>`

Or `brew tap brendan-nasa/brew` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "brendan-nasa/brew"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
